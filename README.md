# projet_stryker_alternance
Modélisation 3D de l'épaule pour la planification chirurgicale. Pipeline complet d'inférence IA (U-Net 3D) pour la segmentation de la scapula et de l'humérus. Génération de maillages STL haute fidélité (Marching Cubes) et validation statistique (Dice Score ~0.91). Inspiré des solutions R&amp;D type Blueprint (Stryker).
